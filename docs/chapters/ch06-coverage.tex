\section{Test Organization Strategy}

\subsection{Coverage Type Grouping (Per Test File)}

Each test file is organized into three main sections:

\subsubsection{1. Statement Coverage Tests [S\#]}
\begin{itemize}
    \item Minimal set of tests to execute all statements
    \item Includes at least one ``happy path'' per function
    \item Includes basic error cases
    \item \textbf{Goal}: Ensure every line of code is executed
\end{itemize}

\subsubsection{2. Branch Coverage Tests [B\#]}
\begin{itemize}
    \item Tests for every conditional branch
    \item Tests for every decision point
    \item Tests for all boolean combinations
    \item \textbf{Goal}: Ensure every if/else path is taken
\end{itemize}

\subsubsection{3. Path Coverage Tests [P\#]}
\begin{itemize}
    \item Tests for every distinct execution path through function
    \item Includes all combinations of decisions
    \item Includes error paths and edge cases
    \item \textbf{Goal}: Ensure all possible flow combinations are tested
\end{itemize}
